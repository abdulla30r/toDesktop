# ToDesktop - Tailwind Project

A modern, responsive website built with **Tailwind CSS** for the ToDesktop platform.

## Getting Started

### Prerequisites
- Node.js and npm installed

### Installation

1. Clone the repository
2. Install dependencies:
```bash
npm install
```

3. Build the Tailwind CSS:
```bash
npx tailwindcss -i ./style.css -o ./output.css --watch
```

4. Open `index.html` in your browser or use a local server:
```bash
npx http-server
```

## Project Structure

- **index.html** - Main HTML file with responsive navigation and content
- **output.css** - Compiled Tailwind CSS (auto-generated)
- **style.css** - Source CSS file
- **tailwind.config.js** - Tailwind configuration
- **script.js** - JavaScript for interactive features
- **assets/** - Images and logos used in the project

## Features

- Responsive navigation bar with mobile menu
- Clean, modern design using Tailwind CSS
- Mobile-first approach
- Font Awesome icons integrated
- Interactive JavaScript functionality

## Technologies Used

- **Tailwind CSS** - Utility-first CSS framework
- **HTML5** - Semantic markup
- **JavaScript** - Client-side interactivity
- **Font Awesome** - Icon library
